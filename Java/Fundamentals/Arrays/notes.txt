    Most Important methods to remember:
    
    import java.util.Arrays; // most impotant thing to do.
    
    arr.length – Returns the total number of elements in the array.
    
    Arrays.toString() – Converts the entire array into a readable string format.
    
    Arrays.sort() – Sorts the array in ascending order.
    
    Arrays.binarySearch() – Searches for an element in a sorted array and returns its index.  If the element is not found, it returns a negative value.
    
    Arrays.equals() – Compares two arrays and returns true if both arrays have the same 
    elements in the same order; otherwise, it returns false.
    
    Arrays.fill() – Fills every element of the array with the specified value.
    
    Arrays.copyOf() – Creates a complete copy of the original array.
    
    Arrays.copyOfRange() – Copies elements from a specified range of the array. The start index is included, and the end index is excluded.

    e.g:
    
    arr.length
    Example: System.out.println(arr.length);

    Arrays.toString()
    Example: System.out.println(Arrays.toString(arr));

    Arrays.sort()
    Example: Arrays.sort(arr);

    Arrays.binarySearch()
    Example: int index = Arrays.binarySearch(arr, 30);

    Arrays.equals()
    Example: System.out.println(Arrays.equals(arr1, arr2));

    Arrays.fill()
    Example: Arrays.fill(arr, 100);

    Arrays.copyOf()
    Example: int[] copy = Arrays.copyOf(arr, arr.length);

    Arrays.copyOfRange()
    Example: int[] range = Arrays.copyOfRange(arr, 1, 4);
