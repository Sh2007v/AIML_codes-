HTML TAGS
    Syntax: <tagname>Content</tagname>
    Examples:   <h1>My First Heading</h1>
                <p>My First Paragraph</p>
    EMPTY HTML TAGS
        :tags with no containing data.
        e.g. : <br>
    NESTED  HTML TAGS
        :tags inside tags

HTML Attributes
    Syntax: <tagname attribute="value">Content</tagname>
    example: <p style="color:red;">Hello</p>
    1.href : <a href="https://www.google.com">Visit Google</a>
    2.src : <img src="image.jpg">
    3.alt : <img src="image.jpg" alt="Nature Image">
    4.style : <p style="color:red;">This is a red paragraph.</p>
    5.lang : <html lang="en"> or <html lang="en-US">
    6.title : <p title="I'm a tooltip">Hover over me</p>  provides extra info about the elem.

HTML Headings
Decreasing importance :
        <h1>Heading 1</h1>
        <h2>Heading 2</h2>
        <h3>Heading 3</h3>
        <h4>Heading 4</h4>
        <h5>Heading 5</h5>
        <h6>Heading 6</h6>

Heading hierarchy
    <h1> → Main page title (use only one per page)
    <h2> → Main sections
    <h3> → Subsections
    <h4>–<h6> → Further subheadings as needed

Changing Heading Size:
    <h1 style="font-size:60px;">Heading 1</h1>


HTML Paragraphs

            <p>This is another paragraph.</p>

    <pre>: displays text exactly as input in HTML source

HTML Text Formatting
    1.<b>: Text is bold
    2.<strong>: important text
    3.<i>: Italic text
    4.<em>: Emphasized text
    5.<small>: Smaller text
    6.<mark>: Highlighted text
    7.<del>: Deleted Text
    8.<ins>: Inserted text
    9.<sub>: Text is slightly below thr main line.
    10.<sup>: Text is slightly above the main line.

HTML Formatting Elements.
    <b>	Displays bold text without extra importance.
    <strong> Displays important text (usually bold).
    <i>	Displays italic text or alternate voice/mood.
    <em>	Displays emphasized text (usually italic).
    <mark>	Highlights or marks text.
    <small>	Displays text in a smaller font size.
    <del>	Displays deleted text with a strikethrough.
    <ins>	Displays inserted text with an underline.
    <sub>	Displays subscript text (e.g., H₂O).
    <sup>	Displays superscript text (e.g., x²).

HTML Links
    <a>:
        Syntax: <a href="URL">Link Text</a>
        e.g. : <a href="https://www.google.com">Visit Google</a>

    target attribute: 
        specifies where the linked page should be displayed
        _self	Opens in the same tab (default).
        _blank	Opens in a new tab or window.
        _parent	Opens in the parent frame.
        _top	Opens in the full browser window.
    
    Absolute URL : 
        contains complete web address
        e.g: <a href="https://www.google.com">Google</a>

    Relative URL :
        refers to a file within the same website.
        e.g: <a href="about.html">About Us</a>

    Image as a link:
        an image can be made clickable as well.
        e.g:
            <a href="home.html">
            <img src="logo.png" alt="Logo">
            </a>
    
    Email Link :
    Syntax: <a href="mailto:abc@example.com">Send Email</a>

    Button as a link:
    <button onclick="document.location='home.html'">Home</button>

    title attribute:
        displays the meaning when mouse is hovered
        <a href="about.html" title="Go to About Page">About Us</a>

HTML TABLES
    used to display data in form of rows and columns
    <table> : table is created using this.
    <tr> : table row
    <td> : table data
    <th> : table header   (bold+Centered)

    Tag	        Purpose
    <caption>	Adds a title/caption to the table
    <thead>	    Groups the table header
    <tbody>	    Groups the table body
    <tfoot>	    Groups the table footer
    <colgroup>	Groups one or more columns for formatting
    <col>	    Defines properties for columns inside <colgroup>

    Table Border : 
        1. Adding a border
            to use border property on <table>,<th> and<td>
                table, th, td {
                    border: 1px solid black;
                }
        
        2. Collapsed Borders :
            table borders are double by default. So do this to collapse them.
            
            e.g. :
            table, th, td {
                border: 1px solid black;
                border-collapse: collapse;
            }

        3. Styling tables borders :
            
            e.g: 
            table, th, td {
                border: 1px solid white;
                border-collapse: collapse;
            }

            th, td {
                background-color: #96D4D4;
            }
        
        4. Rounded Borders : 

            e.g:
                table, th, td {
                    border: 1px solid black;
                    border-radius: 10px;
                }
        
        5. Border Styles :
            e.g:
                th, td {
                    border-style: dotted;
                }
            
            common_values:
                solid
                dotted
                dashed
                double
                groove
                ridge
                inset
                outset
                none
                hidden
        
        6. Border Color : 
            e.g:
                th, td {
                    border-color: #96D4D4;
                }
